# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-13

### Added
- Email Sorter - Automatic email categorization
- PDF Extractor - Text extraction from PDF files
- Social Media Poster - Multi-platform posting
- Notion-Google Sheets Sync - Two-way data sync
- README with installation and usage instructions
- Contributing guidelines

### Features
- IMAP support for email clients
- Batch processing for PDFs
- Twitter, LinkedIn, Reddit support
- Custom field mapping for sync
