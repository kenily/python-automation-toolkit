# Python Automation Toolkit

4 powerful automation scripts in one package:

## Scripts Included

### 1. Email Sorter (`email_sorter.py`)
Automatically categorize and sort your emails using AI.
- Input: CSV email export
- Output: Organized email categories

### 2. PDF Extractor (`pdf_extractor.py`)
Extract text from PDF files with ease.
- Input: PDF files
- Output: Extracted text (TXT/JSON)

### 3. Notion-Google Sheets Sync (`notion_sheets_sync.py`)
Bidirectional sync between Notion databases and Google Sheets.
- Keep your data in sync automatically
- Support for multiple databases

### 4. Social Media Poster (`social_poster.py`)
Schedule and post to multiple social platforms.
- Support for Twitter, LinkedIn, and more
- Bulk posting capabilities

## Requirements
- Python 3.8+
- See individual README files for specific dependencies

## Installation
```bash
pip install -r requirements.txt
```

## Usage
Each script can be run independently. Check individual README files for detailed instructions.
